# Primer Punto — Gramática de Atributos + ETDS para Lenguaje SQL CRUD

Este módulo contiene la solución completa al Primer Punto del examen final:

- Diseño de un lenguaje tipo SQL que soporta operaciones CRUD.
- Definición de una Gramática Independiente de Contexto (GIC).
- Diseño completo de un ETDS (Esquema de Traducción Dirigido por la Sintaxis).
- Definición de atributos heredados y sintéticos.
- Reglas semánticas para cada producción.
- Implementación de la función solicitada por el profesor, basada en gramática de atributos.
- Ejemplo de evaluación con flujo de atributos.

Este punto NO requiere ANTLR ni implementación ejecutable; solo diseño formal.

Archivos incluidos:

- `ETDS.pdf` (opcional si se genera)
- `gramatica.sql.gic`
- `atributos.md`
- `funcion_ejecutar.py`
- `ejemplo_trabajado.md`
